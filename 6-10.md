# Bandit Levels 6-10

## Level 7

Wanted to find a file having size `33c`.

Used:

```bash
find / -size 33c
```

### Learned

* `/` means search from the root directory.
* `find` can search the entire system.

---

## Level 8

Wanted to find password next to the word `millionth`.

Used:

```bash
grep -R "millionth" / 2>/dev/null
```

### Learned

* `grep` searches text.
* `-R` searches recursively.
* `2>/dev/null` hides error messages.

---

## Level 9

File contained repeating lines.

Wanted to find the line that appeared only once.

Used:

```bash
sort data.txt | uniq -c
```

### Learned

* `sort` arranges lines.
* `uniq` works best on sorted output.
* `uniq -c` counts occurrences.

---

## Level 10

Used:

```bash
cat data.txt
```

Password was directly readable from the file.

### Learned

* Sometimes the simplest command is enough.
* Always inspect the file contents before trying complex methods.
